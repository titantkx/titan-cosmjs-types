## Maintenance

This section is for maintainers of this repo, not users.

### Getting started

```sh
# Pull external code
git submodule init
git submodule update

# Install dependencies
npm install
```

### Rebuilding types

```sh
npm run codegen # Generate .ts files into ./src
npm run build # Build .js/.d.ts files
```
